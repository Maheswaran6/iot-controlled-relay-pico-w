Project Overview

This project implements a smart IoT-based relay control system using Raspberry Pi Pico W with built-in WiFi. Electrical appliances such as fans and lights can be controlled remotely using a secure web interface.

Problem Statement

Traditional manual switching causes inconvenience and energy wastage. A smart IoT solution is required for remote, efficient, and reliable control of electrical appliances.

Objectives

  • Design IoT-based relay control using Pico W
  • Enable remote ON/OFF switching via WiFi
  • Demonstrate real-time IoT automation

Hardware

  • Raspberry Pi Pico W
  • Relay Module
  • Fan / Light
  • Power Supply

Software

  • MicroPython
  • Thonny IDE
  • Web Interface
