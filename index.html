IoT Controlled Relay using Raspberry Pi Pico W

Live IoT-inspired web interface for remote electrical appliance control

Project Overview

This project implements an IoT Controlled Relay System using Raspberry Pi Pico W with built-in WiFi. Users can remotely switch electrical appliances such as fans or lights through a web interface, demonstrating real-world IoT automation.

Problem Statement

Manual control of electrical appliances leads to inconvenience and energy loss. A smart, low-cost IoT-based system is required for remote monitoring and control.

Objectives

  • Design IoT-based relay control using Pico W
  • Enable remote ON/OFF switching via WiFi
  • Demonstrate practical IoT automation

Hardware

  • Raspberry Pi Pico W
  • Relay Module
  • Fan / Light
  • Power Supply

Software

  • MicroPython
  • Thonny IDE
  • Web Browser Interface

Advantages

Live IoT UI Remote Access Energy Efficient Low Cost
